**ML** (Machine Learning) **cycle**
![[Pasted image 20260609100323.png|400]]

- **ML algorithm:** mathematical method to learn patterns in data
- **ML model:** output derived from algorithms

**3 Components** of ML algorithms
- **Decision process:** makes predictions/classifications based on input data
- **Error function:** evaluates performance, provides feedback
- **Model optimization process:** fine-tunes algorithm, minimize error, improve accuracy

**4 Learning Types** of ML algorithms
- **Supervised:** labeled data
- **Unsupervised:** unlabeled data
- **Semi-supervised:** small portion of labeled data
- **Reinforcement:** reward correct, penalize mistake
