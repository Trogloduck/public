### Tools
[[#Table of contents|Back to the top]]

Collection of specialized tools for *Forensics, Logic Analysis, and Reverse Engineering*

#### Reverse Engineering & Debugging

- **Ghidra**: NSA-developed open-source **reverse engineering** suite
- **x64dbg**: open-source **debugger** for x64/32 binaries
- **OllyDbg**: debugger for **reverse engineering** at assembly level  
- **Radare2**: open-source **reverse engineering** platform 
- **Binary Ninja**: **disassembling** and **decompiling** binaries.  
- **PEiD**: packer, cryptor, and compiler **detection** tool

#### Disassemblers & Decompilers

Breaking malware into a more understandable format

- **CFF Explorer**: analyze and edit Portable Executable (PE) files
- **Hopper Disassembler**: debugger, disassembler, decompiler
- **RetDec**: open-source decompiler for machine code

#### Static & Dynamic Analysis

**Static** analysis: inspecting code without execution
**Dynamic** analysis: code is running

- **Process Hacker**: memory editor and process watcher
- **PEview**: portable executable (PE) file viewer for analysis
- **Dependency Walker**: displaying an executable’s DLL dependencies
- **DIE (Detect It Easy)**: packer, compiler, and cryptor detection tool

#### Forensics & Incident Response

**Forensics**: collection, analysis, preservation of digital evidence from various sources (computers, networks, storage devices, ...)
**Incident Response**: detection, containment, eradication, recovery from cyberattacks

- **Volatility**: RAM dump analysis framework for memory forensics
- **Rekall**: memory forensics in incident response
- **FTK Imager**: disc image acquisition and analysis tools for forensic use

#### Network Analysis

- **Wireshark**: network protocol analyzer for traffic recording and examination
- **Nmap**: vulnerability detection, network mapping tool
- **Netcat**: read and write data across network connections

#### File Analysis

- **FileInsight**: looking through and editing binary files
- **Hex Fiend**: hex editor
- **HxD**: binary file viewing and editing with a hex editor

#### Scripting & Automation

- **Python**: automation-focused on Python modules and tools
- **PowerShell Empire**: Framework for PowerShell post-exploitation

#### Sysinternals Suite

Collection of advanced system utilities designed to help IT professionals and developers manage, troubleshoot, diagnose Windows systems

- **Autoruns**: shows what executables are configured to run during system boot-up
- **Process Explorer**: information about running processes
- **Process Monitor**: Monitors and logs real-time process/thread activity
