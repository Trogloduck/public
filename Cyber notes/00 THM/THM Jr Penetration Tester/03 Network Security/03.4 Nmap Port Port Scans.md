### NSE - Nmap Scripting Engine
[[#Table of contents|Back to the top]]

Uses Lua language but no need to learn Lua: Nmap contains ~600 prewritten scripts
--> use any or a group of those scripts / install other scripts

`--script=default` / `-sC`: use all scripts from default category
`--script "SCRIPT_NAME"`
`--script "PATTERN*"`

Other categories: auth, broadcast, brute, default, discovery, dos, exploit, external, fuzzer, intrusive, malware, safe, version, vuln

A script can belong to several categories
